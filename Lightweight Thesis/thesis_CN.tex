    % abstract
    \begin{abstract_cn}
        本文设计并实现了一个基于深度卷积神经网络（CNN）的高效图像分类系统。该系统利用迁移学习方法加速模型收敛，并结合数据增强技术提升了模型在复杂场景下的泛化能力。经多组对比实验证明，该方法在准确率和推理速度上均优于传统机器学习方法，具有重要的理论研究和实际应用价值。
    \end{abstract_cn}

    \begin{keywords_cn}
        深度学习；图像识别；卷积神经网络；迁移学习；数据增强
    \end{keywords_cn}

    \begin{abstract_en}
        This thesis designs and implements an efficient image classification system based on Deep Convolutional Neural Networks (CNN). The system utilizes transfer learning methods to accelerate model convergence and combines data augmentation techniques to enhance the model's generalization ability in complex scenarios. Extensive comparative experiments demonstrate that this method outperforms traditional machine learning approaches.
    \end{abstract_en}

    \begin{keywords_en}
        Deep Learning; Image Recognition; CNN; Transfer Learning; Data Augmentation

    \section{数据集与预处理}

    我们选用了公开的图像分类数据集，并按 8:2 的比例随机划分为训练集与测试集。所有的输入图像在进入网络前，均被归一化处理。
